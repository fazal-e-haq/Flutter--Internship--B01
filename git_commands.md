### &nbsp;Git Commands



1\. git init – Initialize a new git repository

2\. git clone – Clone a remote repository to local machine

3\. git status – Check current repository status

4\. git add – Stage files for commit

5\. git commit – Save staged changes to history

6\. git push – Upload commits to remote repository

7\. git pull – Download and merge remote changes

8\. git branch – List, create, or delete branches

9\. git checkout – Switch between branches

10\. git merge – Merge one branch into another
